Your friends and you are meeting each week to spend time together. Usually you sit in the bar at Friday's night or go outside the city to the nature at Saturday or play board games at Sunday. You want to simplify the process of gathering people and that's why you have decided to write a program which could automate this process.
You should create class Friend and each friend will be an instance of this class.
Sometimes the circle of friends is changes - new friends appear, the old are disappear (for example - move to the other town). For creating the right connections you should create the Party class with the next methods:

add_friend(Friend(name)) - add friend 'name' to the list of the 'observers' (people, which will get the invitations, when the new party is scheduled).
del_friend(friend) - remove 'friend' from the 'observers' list.
send_invites() - returns the multiline string, with the list of all friends and the message they have got:

'''John has been invited to the party: Friday, Midnight Pub, 9:00 PM
Lucy has been invited to the party: Friday, Midnight Pub, 9:00 PM'''
You should return an empty string ("") if list of the friends (observers) is empty.
In this mission you coulduse the Observer design pattern.

Examples: 

party = Party()
nick = Friend('Nick')
john = Friend('John')
lucy = Friend('Lucy')
party.add_friend(nick)
party.add_friend(john)
party.add_friend(lucy)
party.del_friend(nick)
party.send_invites("Friday, Midnight Pub, 9:00 PM") == '''John has been invited to the party: Friday, Midnight Pub, 9:00 PM
Lucy has been invited to the party: Friday, Midnight Pub, 9:00 PM'''


Input: methods of the Party class and friends.

Output: multiline string with the list of all who have got the invitations and the text of the invitation.

How it is used: For the automate the mailing system of the notifications for the all observers.

Precondition: All friends names will be different.
